What

What will you learn?

Foundation is a responsive front-end framework.

Sites built on Foundation work great on multiple devices including desktop and laptop computers, tablets, and mobile phones because it is a responsive framework that uses CSS media queries and a mobile-first approach. Designers can create unique designs for a variety of screen sizes, ensuring a smooth experience no matter the device.

ï¿¼ï¿¼Is This Workshop For Me?

If you have been using ZURB Foundation for a while or another framework, this workshop is designed to help you breakthrough to the next level. Often people get to a certain level of proficiency of creating basic sites with ZURB Foundation but they lack the knowledge to be able to take the framework and create a website in any style, pixel perfect, based on hifi-mocks, wireframes or the imagination.

This class is designed to teach you each step of the process. How to create clickable wireframes to protype your site and ui and then how to get them looking pixel perfect by refining the interface or matching existing mocks.

At the end of the workshop you should have the skills to prototype sites and apps quickly and then turn those into production ready code.

Prerequisites

You should be comfortable installing ZURB Foundation from the command line and running tools such as compass, grunt, etc. Optionally you can use a product such as CodeKit (Commercial Software) to render out your ZURB Foundation projects from Sass/ Scss to CSS
